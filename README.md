# ColorSystem
HDR generation color system implementation.

# Features
* tristimulus
* colorspace matrix
* OETF/EOTF(sRGB,BT709,BT2084)
* Yxy, Yu'v'
* Delta u'v', E76, E00
* Bradford adaptation
* Popular gamuts(709,2020,P3,S-Gamut3/cine,AdobeRGB,ACES2065,ACEScg)
* Popular illuminants(A,B,C,D50/55/60/65/75,E,F2/7/11)
